Dear Francois

it sort of does this already, though not completed automatised:

You need to run bibtex on the old and new file, and then compare the bbl 
files, e.g. for pdflatex

pdflatex text-old.tex
bibtex text-old
pdflatex text-new.tex
bibtex text-new

latexdiff text-old.bbl text-new.bbl > text-diff.bbl
latexdiff text-old.tex text-new.tex > text-diff.tex

pdflatex text-diff.tex
pdflatex text-diff.tex

should give you the required result.

The wrapper script "latexdiff-vc" with --pdf or --postscript option will 
run this sequence in one go.
